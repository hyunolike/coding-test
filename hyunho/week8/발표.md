### 1.2 풀이과정 
1. 입력
2. 백트래킹 함수
   1. 배열 길이 2이면 가장 큰 수 answer에 담기
   2. 인덱스 1부터 길이 - 1 만큼 반복
      1. 배열 복사 
      2. 해당 i번 인덱스 제거
      3. 백트래킹 함수(새로운 배열, 해당 인덱스 부분 애너지 양 계산) - 재귀
3. 출력

### 1.3 소스코드

```python
def dfs(arr, _sum):
  global answer

  if len(arr) == 2:
    answer = max(answer, _sum)
    return
  
  for i in range(1, len(arr) - 1):
    new_arr = arr[:]
    del new_arr[i]
    dfs(new_arr, _sum + arr[i-1]*arr[i+1])


answer = 0
n = int(input())
arr = list(map(int, input().split()))
dfs(arr, 0)
print(answer)
```
